# SSLP
<b>S</b>SLP<br />
<b>S</b>ecure<br />
<b>L</b>ucifer<br />
<b>P</b>rotocol<br />


### Build ###
```shell
make
```

Client or Server only
```shell
make client
```
```shell
make server
```
Clean
```shell
make clean
```
### Usage ###
Server
```shell
./server_bin port
```
Client
```shell
./client_bin ip port
```
(If local use 127.0.0.1)
